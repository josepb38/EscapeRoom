# Escape Room JavaScript: La cámara de los cinco desafíos

## Descripción

Aplicación web tipo escape room donde el usuario debe superar cinco niveles consecutivos para recuperar el acceso a una base de datos crítica de una ciudad inteligente.

## Tecnologías

* HTML5
* CSS3
* Bootstrap
* JavaScript
* Geolocalización
* Canvas
* Cámara con `getUserMedia`
* LocalStorage
* Web Workers
* JSON

## Niveles

1. El Guardián de la Ubicación: valida la ubicación del usuario mediante la API de geolocalización.
2. El Cartógrafo Perdido: dibuja un mapa en Canvas y marca la posición obtenida.
3. La Evidencia del Explorador: activa la cámara, captura una fotografía y la guarda en LocalStorage.
4. El Núcleo de Procesamiento: procesa registros simulados con un Web Worker y muestra estadísticas.
5. El Portal Cuántico: procesa un volumen mayor de datos, filtra registros inválidos y permite descargar resultados en JSON.

## Ejecución

1. Abrir la carpeta en Visual Studio Code.
2. Ejecutar el proyecto con Live Server o un servidor local.
3. Abrir la dirección generada en el navegador.
4. Completar los niveles en orden.

La cámara, geolocalización y Web Workers deben probarse desde `localhost` o un contexto seguro.
